CVS hierarchy explained
=======================

./ac-lib3       - network-oriented library (refer to ac* servers)
./devel         - contributed code
./docs          - documentation
./docs/help   	- ???
./docs/install  - ???
./docs/license 	- legal information (1)
./docs/papers   - Papers and other documents
./docs/utils    - Tools to build documentation (2)
./lib	        - library source
./linux         - SPF/Linux library (3)
./samples       - imported source code (4)
./spf3-src      - previous SP-Forth version kernel source (5)
./spf4root      - should be removed
./src           - SP-Forth kernel source
./tools         - tools to create documentation and binary package
                "Инструменты для сборки дистрибутива и оформления документации"
