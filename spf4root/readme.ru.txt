
                                SP-Forth для Windows


ОПИСАНИЕ

  SP-FORTH это надёжная и удобная форт система генерирующая оптимизированный
машинный(native) код для процессоров Intel x86. Работает на Windows 9x, NT
(существуют также порты на Linux, Kolibri OS). SP-Forth распространяется на
условиях GNU General Public License. Детали в файле docs\license\gpl.ru.txt


УСТАНОВКА

  SP-Forth для Windows распространяется как RAR архив или самоустанавливающийся
исполняемый файл.
  Если вы устанавливаете из архива - распакуйте его в желаемую папку. Всё,
теперь он готов к использованию. Если же вы хотите ассоциировать *.f и *.spf
файлы с spf4.exe - запустите docs\install\install.bat, который покажет GUI
окошко для настройки реестра. Или вы можете вручную отредактировать файл
\docs\install\spf_path_install.reg (укажите правильный путь к spf4.exe) и
запустите его. Теперь вы можете писать свой код, сохранить его в *.f файл и
запустить на исполнение двойным кликом мышью.
  Если у вас SFX инсталлятор - просто запустите его. Мастер настройки проведёт
вас через установочный процесс.
  Смотрите файл docs\whatsnew.txt с историей изменений.

  Каталоги:

 \devel   - дополнительные библиотеки и примеры
 \docs    - документация
 \lib     - стандартные библиотеки, обычные ANS и не-ANS расширения такие как float.f, locals.f ...
 \samples - отлаженные GUI и консольные примеры.
 \src     - полные исходники с комментариями и compile.bat скриптом

  Файлы в корневом каталоге:

 envir.spf        - этот файл определяет результат запроса словом ENVIRONMENT?
 help.fhlp        - подключается по умолчанию расширением lib/ext/help.f
 jpf375c.exe      - более старая версия СПФ для пересборки
 spf.err          - этот файл используется СПФ для того чтобы показать
                    текстовое описание ошибки (THROW). Если он отсутствует - в случае исключения
                    вы будет видеть только числовой код
 spf.eng.err      - тот же файл на английском языке
 spf4.exe         - SPF ;)
 spf.ini          - вы можете изменить этот файл и скопировать его в spf4.ini
                    который будет подключён SPF'ом при старте


ДОКУМЕНТАЦИЯ

  Смотрите каталог \docs\papers. Большинство документации на русском языке.
Расширение SPF (lib/ext/help.f) добавляет встроенную помощь в интерпретатор
используя fhlp файлы, которые можно конвертировать в HTML для самостоятельного
использования. Если этой документации недостаточно, задавайте вопросы в
рассылке spf-dev, также загляните на Wiki -

    http://wiki.forth.org.ru
